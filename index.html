GET /arsenal

Arguments

  • account — required
  • platform — either pc, ps4, xb1, or swi — optional, defaults to pc

Response

If the response code is 200, the data is valid JSON; either an object or null if the user doesn't exist or did not enable "Share Loadout Information with the Warframe Arsenal".
